Encrypt, Sign, and Upload

Select a file to upload. The system will automatically perform the following actions:

  1. Generate a new, one-time AES-256 key.
  2. Encrypt the file using this AES key (AES-GCM).
  3. Encrypt the AES key using the server's RSA public key (RSA-OAEP).
  4. Create a digital signature of the file's hash (RSA-PSS).
  5. Upload the secure "envelope" containing all cryptographic materials.
