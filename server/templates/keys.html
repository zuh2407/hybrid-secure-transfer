RSA Key Generation

Generating Your RSA Key Pair

This project requires a 2048-bit RSA key pair to function. A Python script is provided to generate this for you.

To generate your keys, run the following command in your project's terminal:

python client/keygen.py

This command will create two files in your /storage/ directory:

  • server_public_key.pem: The public key. Used for encryption.
  • server_private_key.pem: The private key. Used for decryption and signing. NEVER share this file.
Note: You must run this script at least once before starting the Flask server.
